# 🎬 Blog Semântico de Filmes

Este é o quinto projeto do curso **Formação Front-End** ministrado pelo professor Matheus Battisti. Ele foca na utilização de tags semânticas para criar um blog. Optei por fazer um blog sobre filmes.

## 📖 Descrição

Este blog apresenta uma coleção de artigos sobre diversos filmes, categorizados por gênero e temas relacionados.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3

## 🗂️ Estrutura do Projeto

O projeto está estruturado da seguinte forma:

- **HTML**: Utilização de tags semânticas para melhorar a acessibilidade e a estruturação do conteúdo.
- **CSS**: Estilização dos elementos para garantir uma experiência visual agradável e responsiva.

## 🏷️ Tags Semânticas Utilizadas no HTML

As principais tags semânticas utilizadas no código HTML incluem:

- `<header>`: Contém o cabeçalho do blog, incluindo o menu de navegação.
- `<nav>`: Define a barra de navegação do blog.
- `<main>`: Contém os artigos principais (posts) do blog.
- `<article>`: Cada post de filme é encapsulado em uma tag `<article>`, que contém informações sobre o filme.
- `<section>`: Utilizado para agrupar conteúdos relacionados, como a barra lateral com a busca, categorias e tags.
- `<aside>`: Utilizado para a barra lateral que contém informações complementares ao conteúdo principal.
